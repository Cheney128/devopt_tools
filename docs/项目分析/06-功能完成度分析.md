## 1. 项目概述

### 1.1 项目定位

本项目是一个**企业级网络设备管理系统**，核心功能是：
- **配置备份**：自动/手动备份交换机配置
- **Git集成**：配置版本控制，推送到GitLab/GitHub
- **命令推送**：批量向设备执行命令

### 1.2 技术架构

| 层级 | 技术栈 |
|------|--------|
| 前端 | Vue 3 + Element Plus + Pinia + Vite |
| 后端 | Python 3.9+ + FastAPI + SQLAlchemy |
| 数据库 | MySQL 5.7 |
| 设备交互 | Netmiko (SSH/Telnet) |
| 版本控制 | GitPython |
| 任务调度 | APScheduler |
