- [x] 类定义：`class BackupService`

- [x] `execute_scheduled_backup()` 方法：执行定时备份，包含日志记录
- [x] 详细的日志记录（使用 `[BackupService]` 前缀）
- [x] 完善的错误处理

### 3.3 backup_scheduler.py 修改要点

- [x] 移除 `_execute_backup` 的 `async` 关键字
- [x] 修改 `_execute_backup` 签名：仅接受 `device_id` 参数
- [x] 修改 `add_schedule` 的 `add_job` 调用：`args=[schedule.device_id]`
- [x] `_execute_backup` 内部创建 `SessionLocal()` 会话
- [x] 使用 `asyncio.run()` 调用 `BackupService.execute_scheduled_backup()`
- [x] 添加 1 次重试机制（间隔 5 秒）
- [x] 添加详细的日志记录（使用 `[ScheduledBackup]` 前缀）
- [x] 使用 `try/finally` 确保数据库会话关闭

### 3.4 configurations.py 修改要点

- [x] `collect_config_from_device` API 端点内部调用 `BackupService.collect_config()`
- [x] 保持 API 接口不变
- [x] 移除端点中的业务逻辑，只保留 HTTP 层处理

### 3.5 main.py 修改要点

- [x] 增强 `startup_event` 中的日志记录
- [x] 使用 `logger` 替代 `print`
- [x] 确保异常被正确记录
