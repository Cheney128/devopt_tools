## 3. 详细问题清单

### 3.1 路由配置问题

#### 问题位置
文件: `frontend/src/router/index.js`

#### 问题描述
在路由配置数组中，缺少监控面板的路由配置。需要添加以下路由：

```javascript
{
  path: '/monitoring',
  name: 'monitoring',
  component: () => import('../views/BackupMonitoring.vue'),
  meta: { requiresAuth: true }
}
```

#### 影响分析
- 用户无法通过直接访问URL `/monitoring` 进入监控面板
- 缺少路由会导致组件无法加载

### 3.2 菜单入口问题

#### 问题位置
文件: `frontend/src/App.vue`

#### 问题描述
在侧边栏el-menu组件中，缺少监控面板的菜单项。当前菜单项包括：
- 首页 (/devices/ports/vlans/inspections/configurations/device-collection/git-configs/users)

需要在菜单中添加监控面板入口，建议位置在"配置管理"和"设备采集"之间或单独分区。

#### 影响分析
- 用户无法通过导航菜单发现和使用监控功能
- 与其他功能相比缺少统一入口

---

## 4. 解决方案

### 4.1 解决方案概述

#### 4.1.1 方案一：最小修复（推荐）
仅添加路由配置，让用户可通过URL直接访问。

**优点**：
- 修改量最小
- 快速解决问题

**缺点**：
- 用户难以发现功能入口

#### 4.1.2 方案二：完整修复（推荐）
同时添加路由配置和菜单入口。

**优点**：
- 用户体验最佳
- 功能可被发现

**缺点**：
- 需要修改两个文件

### 4.2 详细修复步骤

#### 4.2.1 步骤1：添加路由配置

**文件**: `frontend/src/router/index.js`

**修改位置**: 在路由数组中，/configurations路由之后

**添加代码**:
```javascript
{
  path: '/monitoring',
  name: 'monitoring',
  component: () => import('../views/BackupMonitoring.vue'),
  meta: { requiresAuth: true }
},
```

**修改前** (行44-49):
```javascript
{
  path: '/configurations',
  name: 'configurations',
  component: () => import('../views/ConfigurationManagement.vue'),
  meta: { requiresAuth: true }
},
```

**修改后**:
```javascript
{
  path: '/configurations',
  name: 'configurations',
  component: () => import('../views/ConfigurationManagement.vue'),
  meta: { requiresAuth: true }
},
{
  path: '/monitoring',
  name: 'monitoring',
  component: () => import('../views/BackupMonitoring.vue'),
  meta: { requiresAuth: true }
},
```

#### 4.2.2 步骤2：添加菜单入口

**文件**: `frontend/src/App.vue`

**修改位置1**: 在模板的el-menu组件中添加菜单项

**添加代码位置**: 在/configurations菜单项之后

```vue
<el-menu-item index="/monitoring">
  <el-icon><MonitorIcon /></el-icon>
  <span>备份监控</span>
</el-menu-item>
```

**修改前** (行124-127):
```vue
<el-menu-item index="/configurations">
  <el-icon><DocumentIcon /></el-icon>
  <span>配置管理</span>
</el-menu-item>
```

**修改后**:
```vue
<el-menu-item index="/configurations">
  <el-icon><DocumentIcon /></el-icon>
  <span>配置管理</span>
</el-menu-item>
<el-menu-item index="/monitoring">
  <el-icon><MonitorIcon /></el-icon>
  <span>备份监控</span>
</el-menu-item>
```

**修改位置2**: 导入MonitorIcon图标组件

**检查**导入语句（行12）是否已包含MonitorIcon：
```javascript
Monitor as MonitorIcon,
```

如果已存在则无需修改，如果不存在则添加。
