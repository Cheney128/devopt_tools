## 一、任务目标

## 二、设备角色规则

| 设备名称模式 | 角色值 | 显示名称 | 标签颜色 |
|-------------|--------|----------|----------|
| `*核心*` | core | 核心 | 红色 (danger) |
| `*汇聚*` | aggregation | 汇聚 | 橙色 (warning) |
| `*接入*` / `*IPMI*` | access | 接入 | 蓝色 (info) |

## 三、执行流程

```
1. 获取设备列表并分类
2. 批量设置设备角色（按页处理）
3. 验证所有设备角色设置成功
4. 生成验证报告
```
